# Contributing to TaskMaster Pro

We welcome contributions from the community and are grateful for your interest in improving TaskMaster Pro.

## How to Contribute

1. **Fork the Repository**
   - Click the 'Fork' button on the top right of the repository page.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/your-username/taskmaster-pro.git
   cd taskmaster-pro
   ```

3. **Create a Feature Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**
   - Follow the project coding style and guidelines.
   - Write tests to cover new functionality.

5. **Commit Your Changes**
   ```bash
   git commit -m 'Add new feature'
   ```

6. **Push to Your Fork**
   ```bash
   git push origin feature/your-feature-name
   ```

7. **Open a Pull Request**
   - Go to the original repository and click on 'New Pull Request'.
   - Ensure your PR description clearly describes the changes made and the problem solved.

## Code Style

- Follow PEP 8 for Python code.
- Use Prettier for formatting JavaScript/TypeScript code.

## Testing Requirements

- All new features must include tests.
- Run the test suite to ensure all tests pass before submitting a PR.
